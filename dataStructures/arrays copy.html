Common Operations

Operation Time Complexity Notes Example Code
insert at the end O(1) Constant No need to iterate; Contiguous storage allows adding w/o index updates arr.push(1)
remove at the end O(1) Constant No need to iterate; Contiguous storage allows adding w/o index updates arr.pop()
insert at the beginning O(n) Linear Requires shifting every element to maintain correct indexes. arr.unshift(3)
remove at the beginning O(n) Linear Requires shifting every element to maintain correct indexes. arr.shift()
access O(1) Constant Directly calculates the memory address of an element using its index. arr[1]
search Worst Case: O(n) Best Case: O(1) Worse case when the element is the last one. Best case when it’s the first element. arr[1]
