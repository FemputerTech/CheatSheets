BUBBLE SORT

Use Cases

  • Useful for small datasets & educational purposes

Steps

  1. Start with first element of the array
  2. Comapre the current element with the next element
  3. if current element > next element, swap
  4. Continue until end of the array
  5. Repeat until no more elements need to be swapped

Time Complexity

  • Worst Case: O(n²)
  • Average Case: O(n²)
  • Best Case: O(n) (when already sorted)

Space Complexity

  • O(1) (in-place)

Advantages

  • Simple and easy to implement
  • No additional memory required

Disadvantages

  • Inefficient for large datasets
  • Not used in real world applications
